# Al Saqi Mineral Water Factory

Bilingual English/Arabic marketing site for Al Saqi natural mineral water.

## Product range

- 330 ml
- 750 ml
- 1200 ml

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

The site is built with Astro and uses a lightweight TypeScript layer for language switching, navigation, reveal motion and the interactive bottle-size selector.

## Brand sources

The visual direction, product details and copy are derived from the two Alsaqi PDFs supplied with the project. The production assets rendered from those documents live in `src/assets/alsaqi-hero.png` and `src/assets/alsaqi-folder-cover.png`.
